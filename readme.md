# Interactive Quiz Application (React.js)

```
This project is an interactive quiz application that guides users through a series of multiple-choice questions and provides a detailed performance summary at the end. It tracks selected, skipped, correct, and incorrect answers in real time, offering clear visual feedback and a structured results breakdown to enhance learning and user engagement.

Built with React.js, leveraging functional components, hooks, and component-based architecture, the application ensures efficient state management and smooth user interactions.

```
